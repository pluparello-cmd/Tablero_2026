Conectar Gemini IA
Ingresá tu API Key de Google Gemini para habilitar el análisis inteligente del tablero.
¿Cómo obtener la API Key?
1. Entrá a aistudio.google.com/apikey
2. Hacé clic en Create API Key
3. Copiá la clave y pegala acá abajo
La clave se guarda solo en tu navegador, no se envía a ningún servidor.
